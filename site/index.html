BBSWS Seeder

Browser-assisted toolkit for seeding Blackbaud school site content. This repository includes login/session bootstrap scripts, seeder scripts by content type, and a local web runner.

View Repository Read Setup Guide

Run Local Web UI

Start with npm run web and open http://localhost:4310.

Login Flow

Use npm run login -- --school <schoolname> to create auth state.

Seed Content

Use scripts such as npm run seed:text -- --school <schoolname>.

• GitHub Pages hosts this static overview only. Seeding and automation still run locally.

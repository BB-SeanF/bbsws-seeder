Runner Help and Legend

This tool wraps the CLI scripts so login and seeding can be run without terminal commands.

School: Subdomain only, for example sisacademy.
Types: Optional comma list to limit run scope. Empty runs all types.
Timeout (ms): Per-action Playwright timeout. Lower is faster but less tolerant.
Default mode: show browser off (headless), pre-check off, dry-run off.
pre-check: Searches for existing categories and skips duplicates.
show browser (headed): When checked, runs with visible browser windows.
dry-run: Validates selectors and flow without creating content.
Start Login: Opens BBID login flow in browser.
Complete Login: In the login modal, sends Enter to the waiting login script after you sign in.
Run Seeder: Runs seed-all with current options.
Stop Seeder: Sends stop signal to active process. May take 2-3 seconds to terminate fully.
Status badges: idle (no run), running (active process), completed, failed, stopped (user cancelled).
Recent Jobs: Click any row to view full logs for that job.
